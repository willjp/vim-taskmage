Projects
========

Like git, taskmage uses a directory to indicate a project-root,
and store completed task-data. 


.. code-block:: python

    /todos/
        home/
            family.mtask
            sideprojects.mtask
        today.mtask

.. code-block:: python

    /todos/.taskmage/
        home/
            family.mtask
            sideprojects.mtask
        today.mtask
